安装LM Studio

LM Studio是一款本地LLM运行工具。

点击LM Studio 进行下载，选择本系统版本。

安装完毕后可以进入设置（右下角齿轮图标），将界面语言修改为中文，如下：

下载大模型

以最近爆火的deepseek-R1为例：

在下载前，先更改模型路径，否则会占用C盘空间，修改为需要的路径即可：

点击左侧搜索图标，在搜索框中输入deepseek r1，就会出现相应的模型，作者的GPU为4070 super，下载14B模型（也是作者最常用的），32B虽然也可以运行，但是生成token速度太慢，无法真正使用。

找到需要下载的模型，点击右下角进行下载，由于作者已经下载，因此显示去使用。

如果无法使用魔法，那么只需要将LM Studio中的下载地址修改为镜像地址即可，在软件中并未提供设置，因此需要手动将LM Studio的运行文件进行修改，操作步骤如下：

  1. 关闭LM Studio

  2. 打开LM Studio安装路径，分别找到两个文件：

    \resources\app.webpack\main\index.js

    \resources\app.webpack\renderer\main_window.js

  3. 将文件中的huggingface.co替换为hf-mirror.com

使用大模型

选择大模型

下载完毕后，进入聊天界面，在顶部选择大模型：

参数配置

上下文长度：吃显存

GPU卸载：吃显存，作者使用14B对于4070 super可以拉满，不过拉满后显存也要告警了，将将能够完全加载。

CPU线程池大小：如果此时没有其他大应用运行的话，可以拉满，这里横条的大小是参考CPU核数的，实际上可以设置为实际CPU线程数，在输入框中自行输入即可。

保持模型在内存中：建议开启，否则CPU会读取硬盘上的模型，会大大降低token生成速度。

尝试mmap()：建议开启，加快模型加载速度。

快速注意力：可以开启，能够稍微加快一点速度，但不多。

Hello World

现在就能愉快的与AI进行对话了。

其他配置

系统提示：可以进行预设，使AI能够按照该预设提示词进行回答，例如你是一只喵娘。。。

温度：日常使用设置0.8即可，如果需要更有创造性，例如写小说等，可以修改为0.9。

其他配置保持默认即可。

安装AnythingLLM

AnythingLLM提供了额外的功能，主要是为LLM提供知识库。

点击AnythingLLM 进行下载，同样选择本系统版本。

运行LLM API

在使用之前，先打开LM Studio，左侧第二个页签，同聊天时一样，在顶部选择加载的模型，加载完毕后点击左上角的开关，让API跑起来，跑起来后可以在日志中看到如下输出：

配置

打开AnythingLLM，会进入教程配置，一步步配置即可，首先会让你选择LLM，选择LM Studio，此时会自动发现端口，选择一下模型然后下一步即可，接下来点击下一步，然后选择下方的Skip Survey，或者填写也可以，接着输入工作区的名称，点击下一步后就来到了AnythingLLM的主界面。

在设置（左下角扳手图标）中配置Embedder首选项，配置成text-embedding-nomic-embed-text-v1.5：

返回主界面，接下来点击左侧工作区列表的对应工作区的上传图标，出现以下界面：

按照提示，上传需要的文档，上传后左侧会出现，选择后点击Move to Workspace，就会出现在右侧，最后点击Save and Embed，稍等片刻（第一次会比较慢，需要几分钟），完成后，此时AI已经就“记住”了你上传的文档，这时聊天会首先参考上传的文档即知识库进行回答。

作者上传了一篇关于ARP协议的文档，问了ARP协议的相关问题，都能很好的从文档中进行总结回答：

如果你不想AI不懂也乱回答，可以在工作区设置将聊天设置中的聊天模式修改为查询，这样，它只会根据你上传的文档进行回答，若查找不到答案，则会返回预设的错误提示。
